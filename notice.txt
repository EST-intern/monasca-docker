Main Open Source Software

Apache Kafka
V0.9.0.1
A distributed publish-subscribe messaging system.
The CMM Message Queue is based on Kafka.
Apache 2.0


Apache Storm
V1.1.1
An open-source distributed realtime computation
system for consuming streams of data and
processing them in arbitrarily complex ways. The
CMM Threshold Engine is based on Apache Storm.
Apache 2.0



Elasticsearch
V2.4.6
An open-source application that provides a highly
scalable full-text search and analytics engine. CMM
uses Elasticsearch as the underlying technology for
storing, searching, and analyzing log data.
Apache 2.0



Grafana 
v4.1.2
An open-source application for visualizing
large-scale measurement data. CMM integrates with
Grafana for visualizing the CMM monitoring data.
Apache 2.0


InfluxDB 
V1.3.4
An open-source database specifically designed to
handle time series data with high availability and
high performance requirements. CMM uses an
InfluxDB database for storing metrics and the alarm
history.
MIT License



Kibana 
V4.6.3
An open-source analytics and visualization platform
designed to work with Elasticsearch. CMM integrates
with Kibana for visualizing the CMM log data.
Apache 2.0



MySQL
V5.5
Relational database system developed as
enhancement and drop-in replacement for MySQL.
CMM uses a MariaDB database for storing
configuration information, alarm definitions, and
notification methods.
GPL V2



Zookeeper
V3.4.11
A open-source service for maintaining configuration
information, naming, providing distributed
synchronization, and providing group services. CMM
uses Zookeeper for administrating the Message
Queue.
Apache 2.0



Memcached
V1.5.3
An open-source distributed memory object caching
system. CMM uses Memcached for dynamically
alleviating the data load that is processed by the log
management components.
BSD 3


Logstash 
V2.4.1
An open-source application that provides a data
collection engine with pipelining capabilities. CMM
integrates with Logstash for collecting, processing
and outputting logs.
Apache 2.0
